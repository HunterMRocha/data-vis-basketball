Data Analytics in Basketball - Alpha Release

Alpha Release

Description:

Basketball statistics are typically analyzed and presented as summaries, either of a game, season, player, or some entity. My analysis begins from the observation that basketball is a game of context. The same outcome on a given play can have radically different meanings for the outcome of a game depending on the context. I will seek to understand on a more contextual level by visually analyzing the game. Specifically, I want to be able to answer the following:
  1. What is the most efficient way to run an offense?
  2. Where do players shoot their most efficient shot?
  3. Can we visualize a trend(s) in the way teams run their offenses?
By answering these questions users will learn how data analytics can be the difference between winning and losing. By visualizing different offenses users will be able to compare different players and teams against each other, and how they stack up against one another. These objectives are going to be met in my features section because users will be able to visualize individual players shot selection -- in addition to teams as a whole. By isolating a player's shot selection users will be able to visualize different strengths and weaknesses in an offense. Users will also be able to visualize any trends, if any, in a players shot selection.
